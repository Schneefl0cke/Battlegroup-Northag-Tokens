# Battlegroup-Northag-Tokens
Unit tokens for Battlegroup Northag or similar tabletop games.

These tokens are designed for 20x20 bases but can be scaled up if needed. They are ideal for use in Tabletop Simulator or physically by printing on colored paper, laminating, and gluing onto an MDF or plastic base for the best quality.

They are not meant to replace painted miniatures, but they are great if you want to try the game, don’t have time, or lack storage space. I use them together with 2D terrain as a travel tabletop that fits in a small box.
